Project Overview

Turning Point is a spiritual and emotional wellness platform designed to help individuals find encouragement through Scripture.

Every emotion card contains a carefully selected Bible verse that addresses specific emotional challenges such as anxiety, loneliness, grief, stress, depression, fear, uncertainty, guilt, and hopelessness.

The purpose of this project is to create a digital sanctuary where users can experience comfort, peace, and emotional healing through God's Word.
